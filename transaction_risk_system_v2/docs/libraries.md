# Используемые библиотеки проекта

В проекте используется следующий набор библиотек, обеспечивающий
сквозной процесс обработки данных: от загрузки и анализа до моделирования,
API и пользовательского интерфейса.

---

## Базовая работа с данными

### numpy
Используется для численных вычислений, работы с массивами и матричными
операциями. Является базовой библиотекой для большинства ML-алгоритмов.

### pandas
Основной инструмент для работы с табличными данными:
загрузка CSV/Excel, предобработка, агрегации, фильтрация,
подготовка данных для анализа и обучения моделей.

### scipy
Используется для статистического анализа:
проверка распределений, вычисление статистических метрик,
работа с вероятностными моделями.

---

## Визуализация и EDA

### matplotlib
Базовая библиотека для построения графиков и визуального анализа данных.

### seaborn
Надстройка над matplotlib, упрощающая построение статистических графиков
(распределения, корреляции, boxplot и т.д.).

### plotly
Используется для интерактивных графиков и визуализаций,
применяется в аналитических дашбордах.

---

## Машинное обучение

### scikit-learn
Основная библиотека машинного обучения.
Используется для:
- кластеризации (KMeans, DBSCAN и др.)
- классификации и регрессии
- оценки качества моделей
- разбиения данных на выборки

### joblib
Используется для сохранения и загрузки обученных моделей
(сериализация моделей).

### catboost
Градиентный бустинг, применяемый для повышения качества
классификации и регрессии, особенно на табличных данных.

---

## Временные ряды и прогнозирование

### statsmodels
Используется для анализа временных рядов и статистического моделирования,
построения прогнозов и анализа сезонности.

---

## Работа с базами данных

### sqlalchemy
ORM и инструмент для работы с базами данных.
Используется для:
- создания структуры базы данных
- загрузки данных из pandas
- выполнения SQL-запросов

> SQLite используется как встроенная СУБД Python (sqlite3 входит в стандартную библиотеку).

---

## Работа с внешними источниками

### requests
Используется для получения данных из внешних API
и загрузки дополнительных наборов данных.

---

## Дашборды и пользовательский интерфейс

### streamlit
Используется для разработки интерактивных дашбордов:
визуализация метрик, фильтрация данных, анализ транзакций.

---

## API

### flask
Лёгкий фреймворк для разработки REST API,
используется для предоставления доступа к модели через HTTP-запросы.
    # pip install flask joblib pandas numpy scikit-learn


### fastapi
Современный фреймворк для разработки API с автоматической документацией,
поддержкой типизации и высокой производительностью.

### uvicorn
ASGI-сервер для запуска FastAPI-приложений.

### pydantic
Используется для валидации входных и выходных данных в API.

---

## Конфигурация и утилиты

### python-dotenv
Используется для хранения и загрузки параметров конфигурации
(пути, настройки, переменные окружения).

---

## Jupyter-среда

### jupyter
Среда для работы с ноутбуками, используется для
исследовательского анализа и экспериментов.

### ipykernel
Обеспечивает работу Python-ядра в Jupyter-ноутбуках.
