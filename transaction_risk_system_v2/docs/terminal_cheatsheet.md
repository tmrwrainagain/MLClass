
# ШПАРГАЛКА ПО РАБОТЕ С ТЕРМИНАЛОМ (macOS / Linux)


Терминал используется для навигации по папкам проекта, поиска файлов,
установки зависимостей и запуска приложений.

Текущая директория (где я сейчас нахожусь):
pwd

Показать файлы и папки в текущей директории:
ls

Показать файлы и папки с подробной информацией:
ls -la

# Переходы между папками:

Шаг назад (в родительскую папку):
cd ..

Переход в конкретную папку:
cd folder_name

Переход сразу в несколько уровней:
cd folder/subfolder

Вернуться в предыдущую папку:
cd -

Перейти в домашнюю папку пользователя:
cd ~

Автодополнение (очень важно использовать):
Начни вводить имя папки или файла и нажми Tab,
терминал сам дополнит путь и уменьшит ошибки.

Пример:
cd tran + Tab
→ cd transaction_risk_system

Работа с проектом:

Перейти в папку проекта:
cd transaction_risk_system

Перейти в папку с документацией:
cd docs

Вернуться в корень проекта:
cd ..

Поиск файлов:

Найти файл по имени в текущей папке и всех вложенных:
find . -name "app.py"

Запуск файлов и приложений:

Запуск Python-скрипта:
python app.py

Запуск Streamlit-приложения:
streamlit run dashboard.py

Работа с виртуальным окружением:

Выход из виртуального окружения:
deactivate

Минимальный набор команд, который нужно запомнить:

pwd        — где я нахожусь  
ls         — что находится в папке  
cd ..      — шаг назад  
cd folder  — зайти в папку  
cd ~       — домашняя папка  
cd -       — вернуться назад  

Эти команды покрывают 90% задач при работе с проектами:
навигация по папкам, поиск файлов и запуск программ.
